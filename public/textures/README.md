# Textures

Place texture assets here. The CSS references `/textures/paper-linen.jpg` as an optional background
overlay. If the file is absent the CSS degrades gracefully — the solid `--bg-paper` colour (#FAF7F2)
and inline SVG fractal noise in `theme.css` provide the paper feel without it.

To add a real linen texture: drop a tileable `paper-linen.jpg` (≈200 KB) into this directory.
